### 4. `masked_normalize`

实现带 mask 的求和归一化。

推荐接口：

```python
def masked_normalize(
    tensor,
    mask,
    normalize_constant,
    dim=None,
):
```

要求：

- 只统计 `mask == 1` 的位置；
- 若 `dim is None`，对所有维度求和；
- 否则沿指定 `dim` 求和；
- 最后除以 `normalize_constant`。

需要实现测试适配：
